チュートリアル

HTML

HTMLとは！

ウェブページを作るための言語だよ！
ウェブページは、インターネットで見ることができるウェブサイトのページのことで、
HTMLを使って書くことでみんながよく見ているページをつくることができるすごい言語だよ

このモードで遊ぶ！

Java

Javaとは！

Javaは、人気のプログラミング言語でたくさんの人に使われていて、とても便利！
ゲームを作ったり、アプリを作ったり、なんでも作ることができるんだ。
しかも、パソコンでもスマホでも使えるシステムを簡単に作ることができるんだよ。

このモードで遊ぶ！

PHP

PHPとは！

PHPは、ウェブサイトを作るときに使われることが多いよ！
PHPを使ってウェブサイトにアンケートやクイズを作ることもできて、楽しいウェブサイトを作るには欠かせない言語だよ

このモードで遊ぶ！

C#

C#とは！

C#は主にWindowsで使われることが多いよ！
Windows用のアプリやゲームを作るのに適していて、とても使いやすい言語なんだ。
C#は英語に近い言語だから、英語が得意な人にとっては理解しやすいかも！

このモードで遊ぶ！
